### Exercises for SQL 

Select the entire table 

```
Select *
from public.example_table;
```

Select specific columns 
```
Select first_name, last_name
from public.example_table;
```

Select the top rows
```
Select *
from public.example_table
limit 3;
```

Distinct values
```
select distinct first_name
from public.example_table;
```
